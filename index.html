Linea del tiempo

Creación de tecnologías que llevaron al desarrollo de las Aplicaciones Híbridas.

Castillo Leal Ana Bertha.


1980

HTML

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, investigador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de “hipertexto” para compartir documentos.

1994

CSS

CSS significa "Hojas de estilo en cascada", que Hakon Wium Lie creó en 1994. CSS se ofreció como lenguaje de estilo web para hacerlo atractivo, y era la solución que la mayoría de los usuarios de HTML buscaban en ese momento.

1995

JavaScript

Aparece en 1995 y es diseñada por Netscape Communications Corp, Mozilla Fundation. JavaScript es un lenguaje de programación interpretado, que puede ser utilizado por profesionales y para quienes se inician en el desarrollo y diseño de sitios web.

2003

Android

Android Inc. fue fundado en 2003 por Andy Rubin, Rich Miner, Nick Sears y Chris White con el objetivo de desarrollar "dispositivos móviles que están al corriente de la ubicación y preferencias del usuario".

2007

iOs

El primer sistema operativo móvil de Apple se presentó el 9 de enero de 2007, fecha en la que Steve Jobs mostró al mundo el iPhone. Todo lo que hacía parecía mágico: gestos multi-touch, todo extremadamente visual, búsqueda en Internet con Safari y una app para YouTube.

2008

PhoneGap

Adobe PhoneGap es un framework para el desarrollo de aplicaciones móviles producido por Nitobi en 2008, y comprado posteriormente por Adobe Systems. Permite a los programadores desarrollar aplicaciones para dispositivos móviles utilizando herramientas genéricas tales como JavaScript, HTML5 y CSS3.

2010

Angular

AngularJS (comúnmente llamado Angular.js o AngularJS), es un framework de JavaScript de código abierto creado en 2010, mantenido por Google, que se utiliza para crear y mantener aplicaciones web de una sola página.

2010

Windows Phone

Windows Phone es el sucesor de Windows Mobile, desarrollado por Microsoft y basado en el núcleo Windows Embedded CE 6.0. Microsoft mostró Windows Phone por primera vez el 15 de febrero de 2010, en el Mobile World Congress de Barcelona y reveló más detalles del sistema en el MIX 2010 el 15 de marzo

2011

Xamarin

Xamarin es una compañía de software estadounidense, adquirida por Microsoft, fundada en mayo de 2011 por Nat Friedman y Miguel de Icaza. Con un código compartido del lenguaje de programación C#, los desarrolladores de software pueden usar Xamarin para escribir aplicaciones móviles nativas para Android, iOS y Windows, y compartir código a través de múltiples plataformas, incluyendo Windows y macOS.

2011

Cordova

Inicialmente Apache Cordova era conocido como PhoneGap, pero en Octubre de 2011 Adobe anunció oficialmente la adquisición de Nitobi, pasando así PhoneGap al control del gigante del software. Es un framework de código abierto para el desarrollo de aplicaciones para móviles multiplataforma, es decir, con un solo código de aplicación podemos utilizarlo en multitud de plataformas móviles, como Android, iOS o Windows Phone.

2013

ionic

Ionic fue creado por Max Lynch, Ben Sperry y Adam Bradley de Drifty Co. en 2013. Ionic es la plataforma que facilita la creación de aplicaciones móviles con tecnologías web para los programadores web.

2013

React

React fue creado por Jordan Walke, un ingeniero de software de Facebook, se volvió código abierto en la JSConf US en mayo de 2013. Es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página.

2014

Switft

Swift es un lenguaje de programación, creado por Apple, que fue presentado en la WWDC de 2014. Este lenguaje de programación nace como una alternativa mucho más sencilla, intuitiva y potente a Objetive-C. Este lenguaje se caracteriza por ser claro y conciso, reduciendo así los errores en los programas y ayudando a tener una sintaxis mucho más precisa.

2015

React native

Desarrollado por Facebook, React Native, permitió el desarrollo nativo para Android, iOS y UWP con React, esto fue anunciado por Facebook en la React Conf en febrero de 2015 y se volvió codigó abierto en marzo de 2015.

2017

Flutter

Flutter es un SDK de código fuente abierto de desarrollo de aplicaciones móviles creado por Google. Suele usarse para desarrollar interfaces de usuario para aplicaciones en Android, iOS y Web así como método primario para crear aplicaciones para Google Fuchsia.
